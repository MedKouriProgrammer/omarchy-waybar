## This is a minimal waybar for omarchy.

### To Downlowd it:
```
git clone https://github.com/HANCORE-linux/waybar-themes.git /tmp/repo && cp -rf /tmp/repo/config/V1.1/. ~/.config/waybar && rm -rf /tmp/repo && omarchy-restart-waybar
```
